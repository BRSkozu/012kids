## この記事の3つのポイント

**多胎妊娠（双子）の妊娠経過と出産準備**について、日本産科婦人科学会・こども家庭庁・国立成育医療研究センターの情報をもとにまとめました。

- **結論から言うと**：双子妊娠で最も重要なのは「膜性診断」です。DD双胎（二絨毛膜二羊膜）・MD双胎（一絨毛膜二羊膜）・MM双胎（一絨毛膜一羊膜）で管理方針が大きく異なります。妊娠初期にエコーで確認されます。
- **ただし注意点も**：双子の平均出産週数は約36週で、早産リスクが単胎より高いです。NICU（新生児集中治療室）のある施設での出産が推奨されます。
- **対象年齢**：双子を妊娠中の方向け

## 詳しい解説

### 膜性診断が最も重要

双子の管理方針は「膜性」（胎盤と膜の共有状態）によって決まります。

| 膜性 | 胎盤 | 膜 | 頻度 | リスク |
|------|------|---|------|--------|
| **DD双胎**（二絨毛膜二羊膜） | 2つ | 2つ | 約70% | 比較的低リスク |
| **MD双胎**（一絨毛膜二羊膜） | 1つ（共有） | 2つ | 約25% | TTTS等のリスクあり |
| **MM双胎**（一絨毛膜一羊膜） | 1つ | 1つ（共有） | 約1% | 最もハイリスク。臍帯相互巻絡のリスク |

膜性診断は**妊娠初期（10〜14週）のエコーが最も正確**です。この時期を逃すと判断が難しくなるため、双子とわかったら早めに確認してもらいましょう。

### 妊娠週数別の管理

| 週数 | DD双胎 | MD双胎 |
|------|--------|--------|
| 〜16週 | 通常の妊婦健診 | 2週ごとのエコー開始 |
| 16〜24週 | 2〜4週ごとの健診 | 2週ごとのエコー（TTTS監視） |
| 24〜28週 | 2週ごとの健診 | 1〜2週ごとの健診 |
| 28〜32週 | 入院管理の検討開始 | 管理入院が多い |
| 32〜36週 | 管理入院 | 管理入院（早期の帝王切開を検討） |
| 37〜38週 | 分娩（帝王切開が多い） | 36週前後での帝王切開 |

### TTTS（双胎間輸血症候群）とは

MD双胎で起こりうる合併症で、共有する胎盤の血管を通じて、一方の赤ちゃんからもう一方に血液が偏って流れる状態です。

- **供血児（ドナー）**：血液が少なくなり、貧血・発育不全・羊水過少に
- **受血児（レシピエント）**：血液が過剰になり、心不全・羊水過多に

MD双胎の約10〜15%に発症。発見が遅れると重篤化するため、定期的なエコーが重要です。治療法として「胎児鏡下胎盤吻合血管レーザー凝固術」があります。

### 出産方法

| 膜性 | 推奨される出産方法 |
|------|-----------------|
| DD双胎（第1子が頭位） | 経腟分娩も選択肢。ただし帝王切開を選ぶ施設が多い |
| DD双胎（第1子が逆子） | 帝王切開 |
| MD双胎 | 帝王切開が推奨 |
| MM双胎 | 帝王切開が必須 |

日本では双子の約70〜80%が帝王切開で出産しています。

### 管理入院について

双子妊娠では、早産予防のために管理入院する場合が多いです。

- **時期**：DD双胎は30〜32週頃から、MD双胎はもう少し早いことも
- **期間**：出産まで（2〜6週間程度）
- **内容**：安静、ノンストレステスト（NST）、超音波検査
- **費用**：健康保険適用＋高額療養費制度の対象。限度額適用認定証を事前に取得しておく

### 自治体の支援制度

多胎家庭向けの支援が近年拡充されています。

| 支援 | 内容 |
|------|------|
| **多胎妊婦の妊婦健診追加** | 一部自治体で追加健診費用の助成 |
| **産前産後ヘルパー** | 自宅に来て家事・育児を支援。無料〜低額 |
| **多胎育児サークル** | 同じ境遇のママ・パパとの交流の場 |
| **ファミリーサポート** | 一時的な預かり支援。多胎家庭は優先利用できる自治体も |
| **おむつ等の現物支給** | 一部自治体で多胎家庭にオムツ・ミルクの支給あり |
| **タクシー券** | 多胎家庭にタクシー利用券を支給する自治体も |

お住まいの自治体の子育て支援窓口に「多胎の支援はありますか」と問い合わせてみてください。
