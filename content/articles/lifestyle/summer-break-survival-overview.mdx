## この記事の3つのポイント

小学生の夏休みは **40日超** の長期間。共働き家庭にとって最大の運用課題です。学童・お弁当・宿題・体調管理・親の働き方が同時に並走します。

- **結論から言うと**：夏休み運用は「学童」「お弁当」「宿題」「体調」「働き方」の5軸を同時に
- **ただし注意点も**：気候変動で熱中症リスク増、午前・夕方の活動シフトが現実的
- **対象年齢**：6〜12歳（小学生）のお子さんを持つ保護者

## 夏休み運用を構成する5つの軸

### 1. 学童・預け先
- 公設学童（8:00〜18:00 が標準、お弁当持参）
- 民間学童（朝食・夕食・送迎オプションあり）
- 祖父母宅の長期滞在
- サマースクール・キャンプ（短期）

詳細は **23区学童 一覧** ページへ。

### 2. お弁当
- 毎日40日超のお弁当作り負担大
- 作り置き・冷凍食品の活用
- 保冷剤・保冷バッグ必携
- 食中毒リスク特に高い時期

詳細は **夏休みのお弁当戦略** 記事へ。

### 3. 宿題・自由研究
- 学校支給の宿題プリント
- 自由研究・読書感想文・絵日記
- 計画的に進めないと終盤で破綻
- 親の伴走スタンスが重要

詳細は **夏休みの宿題伴走** 記事へ。

### 4. 熱中症・体調管理
- 真夏日・猛暑日で外出制限
- WBGT（暑さ指数）に基づく行動判断
- 冷房・水分・睡眠の管理
- 学童での外遊び制限の現実

詳細は **夏休みの熱中症対策** 記事へ。

### 5. 親の働き方調整
- 夏季休暇のずらし取得
- 在宅勤務・フレックスの活用
- 半休・時間休の活用
- 祖父母・親族・ファミサポのサポート要請

## 年齢別の重視ポイント

### 小1〜小2（低学年）
- 学童での過ごし方が中心
- お弁当・身支度の自立度はまだ低い
- 宿題は親が伴走
- 熱中症リスク特に高い

### 小3〜小4（中学年）
- 学童かつ自由時間も
- お弁当を自分で広げる・片付ける
- 宿題は週単位で計画
- 友達との外出はルール明確に

### 小5〜小6（高学年）
- 留守番もできる年齢
- 学童卒業の家庭も多い
- 宿題・自由研究は自主管理に移行
- 友達同士の外出が増える

## 1日のスケジュール例（学童利用・小1〜小3）

```
6:30 起床・朝食
7:30 身支度・お弁当準備
8:00 学童到着
8:30〜12:00 学童プログラム（工作・遊び・宿題）
12:00 昼食（お弁当）
12:30〜17:00 学童プログラム（自由活動・遊び）
17:00〜18:00 親の迎え
18:00〜19:30 帰宅・夕食準備・夕食
19:30〜20:30 お風呂・自由時間
21:00 就寝
```

## 1ヶ月前の準備チェックリスト

- [ ] 学童申込・確認（必要書類）
- [ ] お弁当用品（保冷剤・保冷バッグ・密閉容器）
- [ ] 夏休み中のスケジュール表（家族で共有）
- [ ] 親の有給・在宅勤務・働き方の調整
- [ ] 祖父母・親族のサポート相談
- [ ] 帰省・旅行の計画
- [ ] 熱中症対策グッズ（水筒・タオル・帽子）
- [ ] 子どもの体調管理（前年度との比較）
- [ ] 自由研究のテーマ候補

## 夏休みの「楽しい時間」も忘れずに

学童・お弁当・宿題・働き方の調整に追われがちですが、夏休みは子どもにとって **特別な思い出を作る時期** です。

- 家族旅行（短期でも）
- 花火・夏祭り
- プール・水遊び
- 自然体験（キャンプ・川遊び）
- 祖父母との時間
- 友達との特別な遊び

完璧を求めず、**家族の余白** を意識した運用がコツです。

## 012.kidsの本音

夏休みは **「乗り切る」** という言葉が出てしまうほど親には重い期間ですが、子どもにとっては年間最大のイベント。**親が完璧でなくても良い** という前提で、家族全員で楽しめる運用を意識してください。

最大の落とし穴は **「準備不足」** と **「気持ちの余裕不足」** です。事前計画と、計画通りにいかないことを織り込んだ柔軟な対応で乗り越えられます。
