出産準備リストを見ると、買うべきものが山のようにあって圧倒されますよね。でも実際には「買ったけど使わなかった」というアイテムも少なくありません。この記事では、本当に必要なものと、後から買い足せばいいものを分けて解説します。

## 出産準備の基本方針

### 最低限から始めるのがコツ

赤ちゃんの好みや生活スタイルは、生まれてみないとわからないことが多いです。

**出産前に準備するもの**: 退院後すぐに必要なもの
**産後1ヶ月以内に追加**: 実際の生活で必要になったもの
**必要に応じて検討**: なくても困らないが、あると便利なもの

### 安全基準の確認ポイント

ベビー用品を選ぶ際は以下のマークを確認しましょう。

| マーク | 意味 | 主な対象商品 |
|--------|------|-------------|
| SGマーク | 製品安全協会の安全基準合格 | ベビーカー、チャイルドシート、ベビーベッド等 |
| STマーク | 玩具安全基準合格 | おもちゃ |
| PSCマーク | 消費生活用製品安全法の基準合格 | ベビーベッド、抱っこひも等 |
| JISマーク | 日本産業規格合格 | 各種製品 |

## カテゴリ別：必要度ランキング

### 【睡眠まわり】

| アイテム | 必要度 | 解説 |
|----------|--------|------|
| ベビー布団セット | ★★★ | 固めの敷布団が必須。大人用の布団はNG |
| 防水シーツ | ★★★ | 吐き戻し・おむつ漏れ対策 |
| ベビーベッド | ★★☆ | ペットや上の子がいる場合は必要。レンタルも◎ |
| ベビー枕 | ★☆☆ | 新生児には不要（窒息リスク）。向き癖対策のドーナツ枕は医師と相談 |
| おくるみ | ★★★ | 退院時にも使用、モロー反射対策にも |
| スリーパー | ★★☆ | 冬生まれなら重宝。布団をかけても蹴飛ばす子に |

**安全な寝具のポイント**:
- 敷布団は**固めのもの**を選ぶ（柔らかいと窒息リスク）
- 掛布団は軽いものを（または使わずスリーパーで対応）
- ベビーベッド内にぬいぐるみやクッションを置かない
- 仰向け寝が基本（SIDS予防）

### 【授乳・ミルクまわり】

| アイテム | 必要度 | 解説 |
|----------|--------|------|
| 哺乳瓶（ガラス/プラ） | ★★★ | 母乳でも搾乳や白湯用に1〜2本は必要 |
| 粉ミルク（小缶） | ★★☆ | 産院と同じメーカーが安心。まず小さいサイズで |
| 哺乳瓶洗浄ブラシ | ★★★ | 哺乳瓶を使うなら必須 |
| 消毒グッズ | ★★★ | 電子レンジ消毒が手軽、薬液消毒もあり |
| 授乳クッション | ★★☆ | あると授乳姿勢が楽。普通のクッションで代用可 |
| 授乳ケープ | ★★☆ | 外出時に便利。大判のストールで代用可 |
| 搾乳器 | ★☆☆ | 必要になってから購入で間に合う |

**哺乳瓶の選び方**:

| 素材 | メリット | デメリット |
|------|---------|-----------|
| ガラス | 傷がつきにくい、清潔を保ちやすい | 重い、割れる |
| プラスチック | 軽い、割れにくい | 傷がつきやすい、劣化しやすい |
| PPSU | 軽い、傷がつきにくい | やや高価 |

| おしりふき | ★★★ | 厚手タイプがおすすめ。大量に使う |
| おむつ替えシート | ★★☆ | ペットシーツで代用可 |

### 【衣類】

| アイテム | 必要度 | 枚数目安 | 解説 |
|----------|--------|---------|------|
| 短肌着 | ★★★ | 5〜6枚 | 1年中使える基本の肌着 |
| コンビ肌着 | ★★★ | 4〜5枚 | 足が分かれて動きやすい |
| ツーウェイオール | ★★★ | 3〜4枚 | ドレスとカバーオールの2WAY |
| ガーゼハンカチ | ★★★ | 10枚以上 | 授乳・入浴・よだれ拭きに万能 |
| スタイ（よだれかけ） | ★★☆ | 3〜5枚 | よだれが出始めてからでOK（3ヶ月頃〜） |
| 靴下 | ★☆☆ | 2〜3足 | 室内では不要。外出時のみ |
| ミトン | ★☆☆ | 1〜2組 | 顔をひっかく子に。爪切りで対応可 |

**新生児の衣類選びのポイント**:
- 素材は綿100%が基本（通気性・吸湿性が良い）
- 前開き（かぶるタイプはまだ着せにくい）
- タグは外側についているものを選ぶ
- 新品は一度水通し（洗濯）してから使う

## 賢い節約テクニック

### レンタルがおすすめのアイテム

短期間しか使わないものはレンタルが賢い選択です。

- **ベビーベッド**: 6ヶ月〜1年程度しか使わないことが多い
- **ベビースケール**: 母乳量を測りたい時期だけ
- **ベビーバス**: 1ヶ月しか使わない
- **A型ベビーカー**: B型に買い替えるなら短期間

### 中古品・おさがりの注意点

| アイテム | 中古OK？ | 注意点 |
|----------|---------|--------|
| 衣類 | ◎ | 洗濯すればOK |
| おもちゃ | ○ | 消毒・安全確認をする |
| ベビーカー | △ | 使用年数・リコール情報を確認 |
| チャイルドシート | × | 事故歴不明、劣化の可能性。新品推奨 |
| 抱っこひも | △ | バックル・ベルトの劣化を確認 |
| ベビーベッド | △ | ネジの緩み・木材の割れを確認 |
| 哺乳瓶 | × | 乳首は消耗品、新品を購入 |

### 100均で揃えられるもの

以下は100円ショップで十分な品質のものが手に入ります。

- ガーゼハンカチ
- 食事用エプロン（使い捨てタイプ）
- おしりふきのフタ
- 離乳食用の小分け容器
- お名前シール
- おもちゃ（ガラガラ、布絵本など）
- ベビーハンガー

## 時期別の買い足しスケジュール

### 出産前（妊娠8〜9ヶ月）に準備

**必ず用意**:
- 肌着・衣類（5〜6枚ずつ）
- おむつ（新生児用1パック）
- おしりふき
- ベビー布団セット
- ガーゼハンカチ（10枚）
- ベビーバス
- ベビーソープ
- 保湿剤
- 爪切り・綿棒
- チャイルドシート（車を使う場合）
- 退院時の衣類

### 生後1ヶ月までに追加

- 哺乳瓶の追加（必要に応じて）
- 抱っこひも
- 1ヶ月健診用の外出着

### 生後3ヶ月頃に検討

- ベビーカー（生活スタイルに合わせて選択）
- スタイ（よだれが増える子に）
- おもちゃ（ガラガラ、歯固め等）

### 生後5〜6ヶ月頃に準備

- 離乳食グッズ（食器、スプーン、エプロン、ブレンダー等）
- ベビーチェア
- ベビーゲート・コーナーガード（はいはい対策）

## 先輩ママの「買ってよかった」「いらなかった」

### 買ってよかったもの TOP5

1. **電動鼻吸い器** — 赤ちゃんの鼻詰まりに大活躍。手動より圧倒的に楽
2. **バウンサー** — ワンオペ時の一時待機に。料理中やトイレの間に安全に座らせておける
3. **おしりふきウォーマー（冬生まれ）** — 冷たいおしりふきで泣かなくなった
4. **スワドル（おくるみ）** — モロー反射で起きてしまう子に効果的
5. **授乳ライト** — 夜間授乳で部屋を明るくしすぎない調光ライト

### いらなかったもの TOP5

1. **ベビー用枕** — 新生児には使わない方が安全
2. **ベビーシューズ（歩く前）** — 可愛いけどすぐサイズアウト
3. **おしゃれ着** — 汚れるので肌着で十分だった
4. **ベビー体重計（購入）** — レンタルで十分、使用期間が短い
5. **高機能すぎるベビーモニター** — シンプルなもので十分だった

## 安全に関する最重要ポイント

### リコール情報のチェック

ベビー用品のリコール情報は以下で確認できます。
- 消費者庁リコール情報サイト
- 各メーカーのホームページ
- 経済産業省の製品安全情報

### 誤飲防止の基準

直径39mm以下（トイレットペーパーの芯を通るサイズ）のものは誤飲の危険があります。赤ちゃんの手の届く範囲にないか定期的にチェックしましょう。

### ベビーベッド・寝具の安全基準

- 柵の間隔は6cm以下（PSC基準）
- マットレスと柵の隙間がないこと
- 組み立て後、ネジの緩みを定期的に確認

## まとめ

ベビーグッズは「最低限からスタートし、必要に応じて買い足す」のが賢い方法です。赤ちゃんの個性や家庭環境によって必要なものは異なります。安全基準をしっかり確認しつつ、周りの意見やネットの情報に振り回されすぎず、自分の家庭に合ったものを選んでいきましょう。
